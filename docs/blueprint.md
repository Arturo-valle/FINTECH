# **App Name**: Fintech Hub CR

## Core Features:

- AI Regulatory Assistant: A regulatory assistant chatbot providing information with clear links/citations and a disclaimer. Powered by a Gemini tool in the backend with Cloud Functions and Firebase AI Logic.
- Semantic Directory Search: AI-powered search using Gemini embeddings to match user queries with organization profiles.
- AI-Driven Matchmaking: AI-driven matchmaking using Gemini tool that provides recommendations for potential partners based on calculated embeddings.
- AI Content Assistant: Generate drafts for news, blog posts, event descriptions, and social media messages.
- Conversational Analytics: Use a Gemini tool to translate natural language questions into data queries and explain results.
- Member Directory: Public and private directory listings with filtering and search capabilities.
- Event Management: Allow creation, registration, and management of events, including public and member-only access.
- Role-Based Access Control: Implement access control based on user roles (guest, member, admin) using Firebase Authentication and Firestore security rules.
- Cloud Firestore Database: Uses Firestore database to save users, organizations, events, and resources.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A), a professional color that conveys trust and stability.
- Background color: Light Gray (#F9FAFB), to ensure content legibility.
- Accent color: Bright Purple (#7C3AED), to complement the deep blue and call attention to calls to action.
- Headline font: 'Poppins', a geometric sans-serif that will be used for headings. Body text font: 'Inter', a grotesque-style sans-serif which will ensure legibility.
- Code font: 'Source Code Pro' for displaying code snippets.
- Modern and professional icon set aligned with the fintech industry.
- Clean and structured layout for easy navigation.
- Subtle animations for transitions and feedback to enhance UX.
